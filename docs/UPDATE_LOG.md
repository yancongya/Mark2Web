# Mark2Web 项目更新日志

## v2.0.0 - 多版本管理和UI改进

### 新增功能
- **多版本管理**: 支持生成和管理多个代码/输出版本
- **版本删除功能**: 从下拉菜单中删除不需要的版本
- **独立下拉菜单**: 顶栏版本选择器、预览选项卡版本选择器和源文件下拉菜单现在使用独立状态
- **固定定位下拉菜单**: 所有下拉菜单使用 `fixed` 定位，不会撑开顶栏

### 修复问题
- 修复了顶栏版本选择器无法使用的问题
- 修复了下拉列表冲突和互相干扰的问题
- 修复了骨架动画卡住的问题，生成完成后自动隐藏
- 修复了下拉列表定位问题，不再需要滚动查看内容
- 修复了点击外部关闭功能，所有下拉列表都能正确关闭

### 技术改进
- 为四个下拉列表添加了独立的状态变量和refs
- 改进了骨架动画逻辑，只在预览选项卡生成时显示
- 优化了UI布局，移除了可能导致溢出的属性
- 增强了用户体验，添加了删除确认对话框

### 文件变更
- `components/ResultViewer.tsx`: 添加独立状态、修复定位、添加删除功能
- `App.tsx`: 实现删除处理函数
- `types.ts`: 更新接口定义

## v1.0.0 - 初始版本

### 核心功能
- **AI驱动的代码生成**: 支持多种LLM模型（Gemini、GPT-4、Groq等）
- **多格式输出**: 生成HTML、React(TSX)、Vue组件
- **文本转UI**: 将Markdown或文本转换为前端代码
- **可视化编辑器**: 点击预览元素进行自然语言修改
- **反向工程**: 从现有代码提取内容或分析布局

### 技术栈
- React 19, TypeScript
- Tailwind CSS
- Google GenAI SDK, OpenAI Compatible REST
- Vite构建工具
