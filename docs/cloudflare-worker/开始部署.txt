  3分钟快速部署（无需安装任何工具）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：打开 Cloudflare
  → 访问 https://dash.cloudflare.com
  → 登录（没有就注册，完全免费）

第2步：创建 Worker
  → 左侧菜单：Workers & Pages
  → 点击：Create Application → Create Worker
  → 名称：xiaomimimo-proxy
  → 点击：Deploy

第3步：粘贴代码
  → 点击：Edit Code
  → 删除所有默认代码
  → 打开本文件夹的 worker.js
  → 全选复制 → 粘贴到 Cloudflare
  → 点击：Save and Deploy

第4步：获取 URL
  → 部署成功后，复制 URL
  → 格式：https://xiaomimimo-proxy.xxx.workers.dev
